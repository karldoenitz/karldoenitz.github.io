简介

Tigo 框架是一款使用go语言开发的基于net/http库的web应用框架。开发人员也可以在Tigo框架中使用net/http库的API。

Tigo 框架可以用来快速开发 API、Web 及后端服务等各种应用，是一个 RESTful 的框架， 主要设计灵感来源于 tornado 框架，但是结合了 Go 本身的一些特性（interface、struct 嵌入等）而设计的一个框架。 Tigo 是免费、开源的软件，这意味着任何人都可以为其开发和进步贡献力量。

安装及示例

  • 安装：   go get github.com/karldoenitz/Tigo/...  
  • 示例： 返回一个简单的 Hello Tigo！

Demo如下： 

package main

import (
    "github.com/karldoenitz/Tigo/TigoWeb"
)

type HelloJson struct {
    TigoWeb.BaseHandler
}

func (helloJson *HelloJson)Get() {
    helloJson.ResponseAsJson(struct {
        Status int    `json:"status"`
        Value  string `json:"value"`
    }{0, "Hello Tigo!"})
}

var urls = map[string] interface {} {
    "/test-json": &HelloJson{},
}

func main() {
    application := TigoWeb.Application{
        IPAddress:  "127.0.0.1",
        Port:       8888,
        UrlPattern: urls,
    }
    application.Run()
}
编译： go build main.go   启动： ./main  

此时访问：http://127.0.0.1:8888/test-json 可以获取如下json数据。

{
    "status": 0,
    "value": "Hello Tigo!"
}
